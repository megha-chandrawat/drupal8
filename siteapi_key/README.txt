Modules extend your site information beyond Drupal core.

What does this module work?
--------------------------------

It will alter the existing Site Information form and add new field of site API key. Also It will 
have a custom url (page_json/{api_key}/{nid}) which will accept the api key entered in site information form and
node ID and will preview the node of page type in json format.

List of resources used if any
--------------------------------
Drupal.org and some other sites as well for writing function used for performing these tasks.
Total time to complete task - 3.5 hours
